# account_anstraction

This project implements Account Abstraction to create smart contract wallets that overcome the limitations of traditional Externally Owned Accounts. Users can enjoy:

- ✅ Gasless transactions
- ✅ Social recovery
- ✅ Batched transactions
- ✅ Custom authentication
- ✅ Daily spending limits

Unlike EOAs, smart accounts are programmable, enabling advanced features like gas sponsorship, account recovery, and more.

1. Clone the repository
git clone <your-repo-url>
cd smart_accounts

2. Install Dependencies
npm install

3. Configure Environment

4. Get Testnet ETH

5. Compile Contracts
npm run compile

6. Deploy to Localhost
npx hardhat node

7. Deploy to Sepolia Testnet
npm run deploy:sepolia
